Es la base de cualquier intercambio de datos en la Web y un protocolo de estructura cliente-servidor. Esto significa que una petición de datos es iniciada por el elemento que recibirá los datos (el cliente), normalmente un navegador web. Así, una página web completa resulta de la unión de distintos subdocumentos recibidos, como, por ejemplo: un documento que especifique el estilo de maquetación de la página web (CSS), el texto, las imágenes, vídeos, scripts, etc.
